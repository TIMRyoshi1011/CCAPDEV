# CCAPDEV Web Application Project
<br> BiteBack is a forum-style system that allows users to share detailed experiences, ask questions, and compare recommendations without having to drift off to multiple apps or or scroll endlessly through unrelated posts. <br>

Execute these before running the server:
- npm i express-handlebars 
- npm i mongoose
- npm i dotenv
- npm i bcrypt
<p></p>
To Run Server: <ins>node app.mjs</ins> <br> <br>
Type <ins>localhost:3000</ins> to browser

<br> Live Website: https://biteback-vf0l.onrender.com/
* The website is run on a FREE version
* After 15 minutes of inactivity, setting up will be around 2-3 minutes
