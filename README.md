# CCAPDEV Web Application Project
GitHub Repository for Final Project in CCAPDEV Course from CCS in DLSU from Term 2 AY 2025-2026
- This Program is made by 4 people as a group requirement for this project
- The Layout of This Webpage is Made in Figma and is Made and Run in Visual Studio Code <br> <br>

BiteBack is a forum-style system that allows users to share detailed experiences, ask questions, and compare recommendations without having to drift off to multiple apps or or scroll endlessly through unrelated posts. <br>

**To Compile**: npm i express-handlebars mongoose dotenv bcrypt <br>
**To Run Server**: <ins>node app.mjs</ins> <br> <br>
Type <ins>localhost:3000</ins> to browser

<br> Live Website: https://biteback-vf0l.onrender.com/
* The website is run on a FREE version
* After 15 minutes of inactivity, setting up will be around 2-3 minutes
